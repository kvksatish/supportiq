🤖 Basjoo AI Widget

这是一个可嵌入的AI客服聊天组件示例。点击右下角的聊天按钮开始与AI助手对话！

功能特性

  • 即插即用，一行代码集成
  • 支持自定义主题色和Logo
  • 自动显示引用来源
  • 响应式设计，移动端友好
  • 支持多语言（自动检测）
  • 错误提示和加载状态

集成方式

在您的网站中添加以下代码：

<!-- 加载Widget脚本 --> <script src="basjoo-widget.js"></script> <!-- 初始化Widget --> <script> new BasjooWidget({ agentId: 'your_agent_id', apiBase: 'http://localhost:8001', themeColor: '#3B82F6', title: 'AI助手', welcomeMessage: '你好！有什么可以帮助您的吗？', position: 'right' }).init(); </script>

配置参数

{ agentId: string, // 必需：Agent ID apiBase: string, // 可选：API地址（默认自动检测） themeColor: string, // 可选：主题色（默认#3B82F6） logoUrl: string, // 可选：Logo URL title: string, // 可选：标题（默认"AI助手"） welcomeMessage: string, // 可选：欢迎消息 language: string, // 可选：语言（auto/zh-CN/en-US） position: string // 可选：位置（left/right） }
